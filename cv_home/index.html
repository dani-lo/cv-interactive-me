CurriculumVitae

interactive CV review

CV reviewing just got interactive: this app allows you to review a candidate cv interactively: you can filter, bookmark, add annotations and load different review sessions, to aid you make the best of your tine spent going through all of the candidate's content - in particularly, jobs, projects, company they works at, and more

Webassembly and Nextjs implementation

Ever wondered how an actual web app built in Rust looks and feels? Now you can see it for yourself - interactiveme has 2 identical (feture and UX wise) implmentations - one in Nextjs (React) and one in Web Assembly (Rust with Yew), see the main menu on the left to open either one. NOTE the 2 implementations do not share user token generation, if you want to load the same actions configurations you will need to manually change the user token in either app (for more on user tokens see above)

  • cv review (web assembly - rust)
  • cv review (nextjs)

No personal data required!

The token system which tracks your actions is anonymous - purely a 5 digit alphanumeric token, with no email or password auth requirements. The token can be customised, replaced, and the persistency logic is transparent (you can control it via app settings)

Bookmark Jobs and Projects

All jobs and projects can be bookmarked: this will enable you to then quickly access the bookmarked items from the sidebar menu

Filters

Easily add filters for specific details within a job or project detail, via the contextual modal available on any filterable item

Annotations

You can annotate your impressions as you review the cv: simply type them in the annotations section of the contextual menu - this is available for any item that can be annotated

Review sessions

Upon visiting interactiveme you will be assigned a random 5 digits token. With any interaction such as filtering, annotating etc, your action will either be auto-persisted under the active token or you will be asked if you wish for changes to be persisted (the behaviour can be tweaked within the settings UI). Persisting your actions under a token allows for some interesting functionality for the enhanced cv review experienc
