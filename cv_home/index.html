Interactive Me

interactive CV review

CV reviewing just got interactive: this app allows you to review a candidate cv interactively; you can filter, bookmark, add annotations and load different review sessions, to aid you make the best of the time spent going through all of the candidate's content - in particularly, jobs, projects, company they worked at, and more

WebAssembly and Nextjs implementation

Ever wondered how an actual web assembly app looks and feels? Interactiveme has 2 identical (feature and UX wise) implmentations - one in Nextjs (React) and one in Web Assembly (Rust with Yew), see the links on this page to view them. NOTE the 2 implementations do not share user token generation, if you want to load the same actions configurations you will need to manually change the user token in either app
